Clip Naming Instructions

Before you export

Rename each clip in FCP with a short shortcode — just the number, plus a letter if the piece has multiple sections.

To rename a clip: click it in the browser or timeline, press Return, type the shortcode, press Return again.

Shortcode format

SituationFormatExample
01Single piece01, 02, 03…
07aPiece with sections07a, 07b, 07c…
08-1Multiple camera angles08-1, 08-2…

Example

A concert with 5 pieces where piece 3 has 3 movements:

Then batch export

  1. Select all clips in the browser (Cmd+A)
  2. Go to File > Share > Master File
  3. Choose a destination folder and export

You'll get files like 01.mov, 03a.mov, 03b.mov — ready to drop into this app.
